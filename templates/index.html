Our Project

We utilized advanced machine learning techniques, specifically the LSTM (Long Short-Term Memory) algorithm, to predict stock market movements. This powerful algorithm excels at capturing intricate patterns in historical data, enabling us to make informed predictions about future market behavior.

Technologies We Used

Machine Learning Models and Algorithms

  • Supervised Learning: Regression models (e.g., Linear Regression, Random Forest, Gradient Boosting) to predict stock prices based on historical data.
  • Deep Learning: Recurrent Neural Networks (RNNs), specifically LSTM (Long Short-Term Memory) networks, which are effective for sequence prediction tasks like time-series forecasting.

Data preprocessing

  • Pandas: Python library for data manipulation and analysis, used for cleaning and transforming raw data.
  • NumPy: Fundamental package for numerical computing in Python, essential for handling arrays and matrices.

Web Development

  • Flask and HTML for creating a user-friendly web interface.

Cloud Development

  • AWS (Amazon Web Services) for robust and scalable web deployment.
